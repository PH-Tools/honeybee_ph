# honeybee_phhvac/ — PH-HVAC data model

The dedicated Passive House HVAC package — the migration target for HVAC formerly in `honeybee_energy_ph`. Registered via `_extend_honeybee_ph_hvac.py`.

## Key modules

- `ventilation.py` — ventilation systems (one ventilator per space — see `../context/decisions/0001-*`).
- `heating.py`, `heat_pumps.py`, `renewable_devices.py` — heating / heat-pump / renewable devices.
- `hot_water_system.py`, `hot_water_devices.py`, `hot_water_piping.py` — DHW systems, devices, piping.
- `ducting.py` — ventilation ducting.
- `fuels.py`, `supportive_device.py` — fuels and supportive devices.
- `_base.py` — shared base class.

## Subfolders

- `properties/` — per-host-object PH-HVAC property classes.
- `_diagrams/` — reference diagrams.
