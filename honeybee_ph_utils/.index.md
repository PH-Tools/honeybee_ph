# honeybee_ph_utils/ — shared utilities

Cross-cutting helpers used by the model packages. No PH model objects live here.

## Key modules

- `color.py` — color maps / preview colors.
- `enumerables.py` — enum helpers.
- `face_tools.py`, `polygon2d_tools.py`, `vector3d_tools.py` — geometry helpers.
- `iso_10077_1.py`, `aisi_s250_21.py` — standards-based calculation helpers (window/thermal-bridge, steel-stud).
- `occupancy.py`, `schedules.py`, `ventilation.py` — occupancy/schedule/ventilation helpers.
- `histogram.py`, `sky_matrix.py`, `preview.py`, `input_tools.py` — misc. utilities.

## Notes
- Unit conversion itself lives in the external `PH-units` dependency, not here.
- Tests: `../tests/test_honeybee_ph_utils/`.
