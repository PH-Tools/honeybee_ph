# tests/ — pytest suite

pytest suite mirroring the package layout. Run with `python3 -m pytest`; coverage target is 100% (`fail_under = 100`).

## Contents

- `test_honeybee_ph/` — core model tests.
- `test_honeybee_energy_ph/` — energy-extension tests.
- `test_honeybee_phhvac/` — PH-HVAC tests.
- `test_honeybee_ph_standards/` — reference-data tests.
- `test_honeybee_ph_utils/` — utility tests.

## Notes
- Each package directory mirrors the module layout of the package it tests.
- Serialization round-trip (HBJSON `to_dict`/`from_dict`) is a core thing under test — see `../context/CODING_STANDARDS.md`.
