# honeybee_energy_ph/ — PH extensions to honeybee-energy

Passive House attributes attached to honeybee-**energy** objects (constructions, materials, loads). Registered via `_extend_honeybee_energy_ph.py`. **HVAC here is mostly deprecated** — new HVAC work goes in `../honeybee_phhvac/`.

## Subfolders

- `construction/` — PH construction/material extensions.
- `load/` — PH load extensions.
- `library/` — reference library objects.
- `hvac/` — **deprecated**, migrating to `honeybee_phhvac`.
- `properties/` — per-host-object property classes.
- `cli/` — command-line entry points.
