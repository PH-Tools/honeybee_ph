Welcome to honeybee-ph's documentation!
=======================================

.. image:: http://www.ladybug.tools/assets/img/honeybee.png

`Passive House <https://passivehouse.com/>`_ extension for `honeybee <https://github.com/ladybug-tools/honeybee-core/>`_

Honeybee-ph adds Passive House (PHIUS / PHI) functionalities to honeybee for
energy modeling and certification.


Installation
=======================================

``pip install -U honeybee-ph``

To check if the installation was successful use ``python -c "import honeybee_ph; print(honeybee_ph)"``
