# Honeybee-PH

Passive House extensions for [Ladybug Tools'](https://www.ladybug.tools/) Honeybee energy modeling framework.

Honeybee-PH adds Passive House (PHI & Phius) data to the Honeybee data model, enabling export to PHPP and WUFI-Passive via [PHX](../phx/).

## Installation

```bash
pip install honeybee-ph
```

## Key Features

- Extends Honeybee `Room`, `Face`, and `Aperture` with PH-specific properties
- Passive House ventilation, certification, and mechanical system data
- Integrates with the Grasshopper visual scripting environment via [honeybee-grasshopper-ph](https://github.com/PH-Tools/honeybee_grasshopper_ph)
