# honeybee_ph_standards/ — reference datasets

Reference Passive House standards data shipped as JSON (bundled via `package-data` in `pyproject.toml`). Consumed by the model packages as defaults/lookups.

## Subfolders

- `constructions/` — reference construction/material data.
- `constructionsets/` — reference construction sets.
- `programtypes/` — PH-relevant program types.
- `schedules/` — reference schedules.
- `sourcefactors/` — primary-energy / CO2 source factors.

## Notes
- Data is JSON, not code — treat these files as reference values, not editable model logic.
- Tests: `../tests/test_honeybee_ph_standards/`.
